# 🔐 Antef Task 6 – Login Page UI

## 📌 Project Overview

This project is a modern and responsive **Login Page UI** built using HTML and CSS. It features a split-screen layout with an image on the left and a login form on the right, providing a clean and user-friendly interface.

---

## 🚀 Features

* 🖼️ Split layout (Image + Form)
* 🔐 Login form with email and password fields
* 🎨 Clean and modern UI design
* 🔘 Styled login button with hover effect
* 🔗 “Forgot Password” and “Register Now” links
* 📱 Responsive design for smaller screens

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)

## ▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Double-click `index.html`
4. Open in any web browser

---

## 🔗 Functionality

* Users can enter email and password
* “Login” button for submission (UI only)
* “Forgot Password” option (UI placeholder)
* “Register Now” redirects to external signup page
